# Customer-Segmentation-Using-Unsupervised-Learning
Key Features:
Exploratory Data Analysis (EDA):

Visualizes distributions of age, income, and spending score

Shows correlations between features

Provides gender distribution analysis

Clustering Process:

Uses Elbow Method and Silhouette Score to determine optimal K

Applies K-Means clustering with feature standardization

Visualizes clusters in 2D (PCA and t-SNE) and 3D

Cluster Analysis:

Provides detailed statistics for each cluster

Identifies customer segments based on spending patterns

Marketing Strategies:

Provides tailored marketing strategies for each segment

Suggests communication channels

Saves results for further analysis
